## Requirement

- Python 3.8.18
- CUDA Toolkit 11.0.3
- cuDNN v8.0.5

## Installation

- OpenCV

  ```bash
  pip install opencv-python
  pip install opencv-contrib-python
  ```

- TensorFlow

  ```bash
  pip install tensorflow==2.4.0
  ```

- Scikit Learn

  ```bash
  pip install scikit-learn
  ```

- Matplotlib

  ```bash
  pip install matplotlib==3.6
  ```

## Usage

- Training Model
  
  ```Bash
  python build_model.py
  ```

- Deploy Model

  ```Bash
  python main.py
  ```
